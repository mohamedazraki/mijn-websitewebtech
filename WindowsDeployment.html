Automated PowerShell Deployment

Dit project is een automatisatieproject waarbij ik PowerShell heb gebruikt om Windows configuraties sneller en gestructureerder uit te voeren.

Over dit project

Voor het opleidingsonderdeel Scripting heb ik een PowerShell-project gemaakt waarmee een Windows Server en Windows-client gedeeltelijk automatisch geconfigureerd worden.

Het script werkt met verschillende configuratiebestanden zoals XML, JSON, CSV en TXT. Op basis daarvan worden instellingen, gebruikers, OU’s en security groups aangemaakt.

Belangrijkste onderdelen

  • Automatisch configureren van Windows Server-instellingen.
  • Werken met externe configuratiebestanden.
  • Aanmaken van gebruikers, OU’s en security groups.
  • Logging voorzien om acties en fouten te kunnen opvolgen.

Gebruikte tools

PowerShell, Windows Server, Windows 11, XML, JSON, CSV, TXT en logging.

Doel van het project

Het doel was om repetitieve installaties en configuraties te automatiseren zodat een Windows-omgeving sneller, correcter en overzichtelijker opgezet kan worden.

Wat heb ik geleerd?

  • Ik heb geleerd hoe ik PowerShell kan gebruiken om beheertaken te automatiseren.
  • Ik heb geleerd hoe ik configuratiebestanden kan uitlezen en gebruiken in scripts.
  • Ik heb geleerd waarom logging belangrijk is bij automatisatie.
  • Ik heb geleerd hoe belangrijk testen is vooraleer een script in een echte omgeving gebruikt wordt.
