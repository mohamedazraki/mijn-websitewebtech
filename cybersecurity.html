Cybersecurity en Medusa

Dit project gaat over cybersecurity, wachtwoordbeveiliging en het testen van systemen in een gecontroleerde labo-omgeving.

Over dit project

Tijdens dit project heb ik gewerkt rond basisprincipes van security testing, vulnerability scanning en wachtwoordbeveiliging.

Ik heb onderzocht hoe brute-force technieken werken met Medusa en waarom zwakke wachtwoorden een groot risico vormen voor systemen en gebruikersaccounts.

Belangrijkste onderdelen

  • Onderzoeken hoe brute-force aanvallen in een labo-omgeving werken.
  • Testen van zwakke wachtwoorden met Medusa.
  • Analyseren waarom sterke wachtwoorden belangrijk zijn.
  • Leren hoe systemen beter beschermd kunnen worden tegen login-aanvallen.

Gebruikte tools

Linux, Medusa, command line tools en security testing technieken.

Doel van het project

Het doel was om beter te begrijpen hoe aanvallen op zwakke wachtwoorden werken en waarom sterke wachtwoorden en goede beveiligingsmaatregelen belangrijk zijn.

Wat heb ik geleerd?

  • Ik heb geleerd hoe belangrijk sterke wachtwoorden zijn.
  • Ik heb geleerd hoe brute-force tools in een gecontroleerde omgeving gebruikt worden.
  • Ik heb geleerd hoe aanvallers misbruik kunnen maken van zwakke loginbeveiliging.
  • Ik heb geleerd waarom monitoring en accountbeveiliging belangrijk zijn.
